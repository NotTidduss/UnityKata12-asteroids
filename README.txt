Unity Kata #12 - Asteroids

What is Asteroids?
A space-themed multidirectional shooter arcade game.
Player controls a space ship and shoots asteroids / meteors / comets... = obstacles
If player ship is hit (too often), game over.
Score if based on obstacles shot.
